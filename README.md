"# alibaixiu_websit" 
# 技术栈(前后端分类模式)
- 前端
  - ajax+jquery+js+html+css+bootstrap
  - art-template、font-awesome、swipe
- 后端
  - node+express框架
- 数据库
  - mongodb

# 项目概述
  - 实现网站内容管理和内容展示
  - 超级管理(管理网站)和普通用户(浏览网站发布评论)
# [项目知识点见个人博客](https://sparkparis.github.io/2020/04/27/Ajax-project/)

# 项目运行
- 安装配置mongodb
- 搭建后台环境
- npm install 安装环境依赖
- node app.js启动后台服务器
